Операторы цикла

Итерация – это повторение одной и той же последовательности действий с целью достижения результата

Она помогает решить задачи, когда одно действие нужно повторить несколько раз: вывести сообщение, получить или записать данные, отправить запрос и т.д.

Одно прохождение цикла – итерация

Итерация помогает контролировать выполнение программы.

Например, если сказать циклу «выполни десять раз» то он выполнит именно столько раз – точно

И наоборот, если не знаем, сколько нам нужно раз – ставим условие для каждой итерации. Когда заданное условие выполнится – он закончится. Так что считать итерации вручную не понадобится.

For

Это управляющая конструкция программного кода, которая нарушает линейность исполнения алгоритма и позволяет выполнить указанный код множество раз. Например, нужно принять 30 капель лекарства

Линейный алгоритм

1.      Приготовить стакан

2.      Открыть крышку

3.      Получить 1 каплю

4.      Получить еше одну

5.      И так далее

34. Закрыть лекарство

35. Принять порцию

Цикл for

1.      Приготовить стакан

2.      Открыть крышку

3.      Получить 30 капель

4.      Закрыть лекарство

5.      Принять порцию

В java цикл for необходим для сокращения кода

For (<начальная точка><условие ><оператор счетчика>) {

 \\ тело цикла

}

While

Оператор потока управление, который позволяет многократно выполнить код на основе заданного логического условия

Цикл for применяется, когда мы знаем кол-во повторений, а while когда нам оно не важно или не известно, и закончится когда выполнится условие

While (num <= 5) {

System.out.print(num);

Num++;

}

Do while

- Оператор цикла, который выполняет итерацию еще до проверки условия, хотя бы один раз, даже если условие ложно. После выполнения проверяется условие – если истинно, то повторно повторяется тело

Do {

                System.out.print(num);

Num++;

} while (num <= 5)
