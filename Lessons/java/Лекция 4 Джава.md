# Лекция №4: Символьные и строковые типы данных. Классы для работы со строками в Java

## Типы данных

- **Символьные**: char
- **Строковые**: String

## Тип char

- Представляет собой 16-битный Unicode-символ.
- Это примитивный тип данных, используемый для хранения отдельных символов, таких как буквы, цифры, знаки препинания и специальные символы.
- Символ char представляет отдельный символ и записывается в одинарных кавычках.
- Примеры инициализации:
    - char symbol1 = 1078; // по индексу символа в таблице UTF-8 (фактически это Unicode code point, соответствующий символу 'ж' в UTF-16, так как char в Java — 16-битный).
    - char symbol2 = 'ж'; // по значению символа.
    - char symbol3 = '\u0436'; // через шестнадцатеричную форму Unicode (это также символ 'ж').

## Тип String

- Строка представляет собой последовательность символов.
- Для работы со строками в Java определён класс String, который предоставляет ряд методов для манипуляции строками.
- Физически объект String — это ссылка на область в памяти, в которой размещены символы.
- Создание строки:
    - С использованием одного из конструкторов класса String.
    - Напрямую присвоением строки в двойных кавычках, например: String str = "Привет";.
- Поскольку строка рассматривается как набор символов, можно применить метод length() для нахождения длины строки или длины набора символов. Например: int len = str.length();.
- Метод toCharArray() позволяет преобразовать строку обратно в массив символов: char[] chars = str.toCharArray();.
- Метод isEmpty() проверяет строку на пустоту и возвращает true, если строка пуста (длина равна 0): boolean empty = str.isEmpty();.
- Переменная типа String может не указывать на какой-либо объект и иметь значение null. Значение null не эквивалентно пустой строке "".
- Пример проблемы с null: если строка null, обращение к её методам (например, null.length()) приведёт к ошибке NullPointerException.
- Чтобы избежать ошибок, рекомендуется проверять строку на null перед использованием методов: if (str != null) { /* код */ }.

## Основные методы класса String

Класс String предоставляет множество методов для работы со строками. Ниже приведён список основных методов с краткими описаниями (на основе стандартных операций в Java, чтобы охватить все возможные аспекты лекции, включая те, что могли быть в изображениях с таблицами методов).

|Метод|Описание|Пример|
|---|---|---|
|charAt(int index)|Возвращает символ по указанному индексу в строке.|"hello".charAt(1) → 'e'|
|concat(String str)|Объединяет текущую строку с указанной строкой.|"hello".concat(" world") → "hello world"|
|contains(CharSequence s)|Проверяет, содержит ли строка указанную последовательность символов.|"hello".contains("ell") → true|
|endsWith(String suffix)|Проверяет, заканчивается ли строка указанным суффиксом.|"hello".endsWith("lo") → true|
|equals(Object anObject)|Сравнивает строки на равенство (с учётом регистра).|"hello".equals("hello") → true|
|equalsIgnoreCase(String anotherString)|Сравнивает строки на равенство без учёта регистра.|"hello".equalsIgnoreCase("HELLO") → true|
|indexOf(int ch)|Возвращает индекс первого вхождения указанного символа.|"hello".indexOf('l') → 2|
|indexOf(int ch, int fromIndex)|Возвращает индекс первого вхождения символа, начиная с указанного индекса.|"hello".indexOf('l', 3) → 3|
|lastIndexOf(int ch)|Возвращает индекс последнего вхождения указанного символа.|"hello".lastIndexOf('l') → 3|
|length()|Возвращает длину строки.|"hello".length() → 5|
|replace(char oldChar, char newChar)|Заменяет все вхождения старого символа на новый.|"hello".replace('l', 'p') → "heppo"|
|split(String regex)|Разделяет строку на массив подстрок по регулярному выражению.|"a-b-c".split("-") → ["a", "b", "c"]|
|startsWith(String prefix)|Проверяет, начинается ли строка с указанного префикса.|"hello".startsWith("he") → true|
|substring(int beginIndex)|Возвращает подстроку, начиная с указанного индекса.|"hello".substring(2) → "llo"|
|substring(int beginIndex, int endIndex)|Возвращает подстроку от beginIndex до endIndex (не включая endIndex).|"hello".substring(1, 4) → "ell"|
|toLowerCase()|Преобразует строку в нижний регистр.|"HELLO".toLowerCase() → "hello"|
|toUpperCase()|Преобразует строку в верхний регистр.|"hello".toUpperCase() → "HELLO"|
|trim()|Удаляет ведущие и trailing пробелы.|" hello ".trim() → "hello"|
|valueOf(Object obj)|Преобразует объект в строковое представление.|String.valueOf(123) → "123"|
|compareTo(String anotherString)|Сравнивает строки лексикографически (возвращает отрицательное, нулевое или положительное значение).|"apple".compareTo("banana") → отрицательное|
|getChars(int srcBegin, int srcEnd, char[] dst, int dstBegin)|Копирует символы из строки в массив.|Используется для извлечения группы символов|
|join(CharSequence delimiter, CharSequence... elements)|Соединяет строки с указанным разделителем.|String.join(", ", "a", "b") → "a, b"|
|isEmpty()|Проверяет, пуста ли строка.|"".isEmpty() → true|
|toCharArray()|Преобразует строку в массив символов.|"hello".toCharArray() → ['h', 'e', 'l', 'l', 'o']|
