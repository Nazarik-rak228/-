# Определение
Компьютерная сеть - это группа компьютеров, соединенных друг с другом  каналами связи. 
# Деление по признакам

Компьютерные сети распределяются по территориальному признаку:
1. Локально-вычислительные сети - сети, которые расположены в одном или нескольких зданиях. Чаще всего такие сети размещаются в рамках одной организации. 
2. Глобальные(распределенные) компьютерные сети - это совокупность локальных сетей, соединенных каналами связи. Также глобальные сети разбиваются на основные 4 вида:
	1. Городские 
	2. Региональные
	3. Национальные
	4. Транс-национальные.

# По степени использования сети делятся на:
1) Коммуникационные - для передачи информации, так же выполняет задачи, связанные с преобразованием данных.
2) информационная сеть - для хранения информации, состоит из информационных систем. На базе коммуникационной сети можно построить несколько информационных сетей.

Информационная система - система поставщик или потребитель  информации

# Преимущества работы в сети
1) Совместное использование информации.
2) Совместное использование аппаратных средств.
3) Совместное использование программных средств.
4) Обмен сообщениями. 
# Недостатки работы в сети
1) Быстрое распространение вирусов.
2) Возможность взлома или утери информации.
