# 1. DOM (Document Object Model)

**DOM** — это объектная модель документа.  
Браузер преобразует HTML в дерево объектов, которыми можно управлять через JavaScript.

DOM позволяет:

- изменять текст и HTML,
    
- менять стили,
    
- добавлять и удалять элементы,
    
- реагировать на действия пользователя.
    

Главный объект DOM — `document`.

---

## 1.1 Объект document

`document` представляет всю HTML-страницу.

### Основные свойства

|Свойство|Описание|
|---|---|
|`document.title`|Заголовок страницы|
|`document.body`|Тело документа|
|`document.URL`|Адрес страницы|
|`document.cookie`|Cookies|

---

## 1.2 Поиск элементов в DOM

### Методы получения элементов

|Метод|Что делает|Возвращает|
|---|---|---|
|`getElementById("id")`|Поиск по id|Один элемент|
|`getElementsByClassName("class")`|По классу|Коллекцию|
|`getElementsByTagName("tag")`|По тегу|Коллекцию|
|`querySelector("selector")`|Первый по CSS-селектору|Один элемент|
|`querySelectorAll("selector")`|Все по селектору|Коллекцию|

---

## 1.3 Работа с элементами

### Изменение содержимого

|Свойство|Назначение|
|---|---|
|`element.textContent`|Изменяет текст|
|`element.innerHTML`|Изменяет HTML|

---

### Изменение стилей

|Свойство|Пример|
|---|---|
|`element.style.color`|Цвет текста|
|`element.style.backgroundColor`|Цвет фона|

---

### Работа с классами

|Метод|Назначение|
|---|---|
|`classList.add()`|Добавить класс|
|`classList.remove()`|Удалить класс|
|`classList.toggle()`|Переключить класс|
|`classList.contains()`|Проверить наличие|

---

## 1.4 Создание и удаление элементов

### Создание и добавление

|Метод|Назначение|
|---|---|
|`createElement()`|Создать элемент|
|`appendChild()`|Добавить в конец|
|`prepend()`|Добавить в начало|
|`insertBefore()`|Вставить перед элементом|

---

### Удаление

|Метод|Назначение|
|---|---|
|`remove()`|Удалить элемент|
|`removeChild()`|Удалить дочерний элемент|

---

## 1.5 Навигация по DOM

|Свойство|Назначение|
|---|---|
|`parentNode`|Родитель|
|`children`|Дочерние элементы|
|`firstElementChild`|Первый ребёнок|
|`lastElementChild`|Последний ребёнок|
|`nextElementSibling`|Следующий элемент|
|`previousElementSibling`|Предыдущий элемент|

---

# 2. События (Events)

Событие — это действие пользователя или браузера.

### Способы обработки

1. Через свойство:
    

element.onclick = function() {}

2. Через addEventListener (рекомендуется):
    

element.addEventListener("click", function() {})

---

## Основные события

|Событие|Когда происходит|
|---|---|
|`click`|Клик|
|`dblclick`|Двойной клик|
|`mouseover`|Наведение|
|`mouseout`|Уход курсора|
|`keydown`|Нажатие клавиши|
|`keyup`|Отпускание клавиши|
|`change`|Изменение значения|
|`submit`|Отправка формы|

---

# 3. Формы

Формы используются для ввода данных.

## Получение формы

|Способ|Пример|
|---|---|
|По id|`document.getElementById("form")`|
|Через коллекцию|`document.forms["formName"]`|

---

## Работа с полями

|Свойство|Назначение|
|---|---|
|`element.value`|Значение|
|`element.checked`|Состояние checkbox|
|`element.disabled`|Блокировка|

---

## Отмена отправки формы

form.addEventListener("submit", function(event) {  
    event.preventDefault();  
});

---

# 4. BOM (Browser Object Model)

**BOM** — объектная модель браузера.  
Если DOM управляет документом, то BOM управляет браузером.

Главный объект — `window`.

---

## 4.1 Объект window

`window` — глобальный объект браузера.

### Методы

|Метод|Назначение|
|---|---|
|`alert()`|Сообщение|
|`prompt()`|Ввод|
|`confirm()`|Подтверждение|
|`open()`|Открыть окно|
|`close()`|Закрыть окно|
|`print()`|Печать|

---

### Таймеры

|Метод|Назначение|
|---|---|
|`setTimeout()`|Один раз через время|
|`setInterval()`|Повтор через интервал|
|`clearTimeout()`|Остановить таймер|
|`clearInterval()`|Остановить интервал|

---

## 4.2 Объект location

Отвечает за адрес страницы.

### Свойства

|Свойство|Назначение|
|---|---|
|`location.href`|Полный URL|
|`location.hostname`|Домен|
|`location.pathname`|Путь|
|`location.protocol`|Протокол|

---

### Методы

|Метод|Назначение|
|---|---|
|`location.reload()`|Перезагрузка|
|`location.assign()`|Переход|
|`location.replace()`|Переход без истории|

---

## 4.3 Объект history

Управляет историей переходов.

|Метод|Назначение|
|---|---|
|`history.back()`|Назад|
|`history.forward()`|Вперёд|
|`history.go(n)`|На n шагов|

---

## 4.4 Объект navigator

Информация о браузере.

|Свойство|Назначение|
|---|---|
|`navigator.userAgent`|Данные браузера|
|`navigator.language`|Язык|
|`navigator.platform`|Платформа|

---

## 4.5 Объект screen

Информация об экране.

|Свойство|Назначение|
|---|---|
|`screen.width`|Ширина|
|`screen.height`|Высина|
|`screen.availWidth`|Доступная ширина|
|`screen.availHeight`|Доступная высота|

---

# Разница DOM и BOM

|DOM|BOM|
|---|---|
|Работает с HTML|Работает с браузером|
|Главный объект — `document`|Главный объект — `window`|
|Изменяет элементы страницы|Управляет окном браузера|

---

# Итог для зачёта

- DOM — работа с документом
    
- BOM — работа с браузером
    
- `window` — главный объект
    
- `document` — часть `window`
    
- Элементы ищем через getElementById / querySelector
    
- События через addEventListener
    
- Формы через value и submit
