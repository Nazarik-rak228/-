| -------------------------------- | --- | ----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |

| Совместимость                    | 70% | Работает на Windows / Linux / macOS. Но на Windows 11 24H2+ и Server 2025 бывают серьёзные проблемы с установкой гостевых ОС.                                                                                                                               |

**Итоговый средний балл** ≈ 57% — "удовлетворительно с натяжкой"

**Приоритетные рекомендации** :

1. **Исправить / упростить работу с современными Windows-хостами** (Hyper-V / VBS / Core Isolation) — сейчас это главная причина лагов и «черепашьего режима» у большинства домашних пользователей.
2. **Сделать Guest Additions более надёжными и автоматическими** — установка/обновление не должна превращаться в случайные ошибки и ручное вмешательство.
3. **Улучшить стабильность GUI-менеджера** — краши при работе со снапшотами и массовым удалением VM до сих пор встречаются.
4. Ввести систему авто обновления программы
